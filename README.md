# DVWA on Raspberry Pi

This project demonstrates how to deploy **DVWA (Damn Vulnerable Web Application)** on a Raspberry Pi for security learning purposes.

## Project Overview

DVWA is a deliberately vulnerable web application used to practice web security testing and exploitation techniques.

## ⚠️ Disclaimer

I am learning cybersecurity and system administration.  
This repository is part of my learning journey while building small labs with Raspberry Pi and vulnerable applications like DVWA.

There may be mistakes or suboptimal practices in this project.
Feel free to open an issue or suggest improvements.



This lab runs DVWA on a Raspberry Pi using a LAMP stack:

- Apache
- MariaDB
- PHP

## Access DVWA

After installation, access DVWA at:
`http://<raspberry-ip>/DVWA`

Default credentials:

admin / password

## Security Note

⚠️ DVWA is intentionally vulnerable.  
Use only in **isolated lab environments**.
